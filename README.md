# Portfolio — Jules Landauer

Site statique HTML/CSS/JS prêt à être mis en ligne gratuitement avec GitHub Pages.

## Structure

- `index.html` : page principale du portfolio
- `assets/css/style.css` : design du site
- `assets/js/main.js` : scripts simples
- `assets/img/` : images et captures
- `assets/docs/` : preuves et documents de projets

## Mise en ligne gratuite avec GitHub Pages

1. Créer un compte GitHub ou se connecter.
2. Créer un nouveau dépôt nommé par exemple `portfolio-jules-landauer`.
3. Envoyer tout le contenu de ce dossier dans le dépôt.
4. Aller dans `Settings` > `Pages`.
5. Dans `Build and deployment`, choisir :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
6. Cliquer sur `Save`.
7. Attendre 1 à 3 minutes. Le site sera accessible à une adresse du type :
   `https://juju2303.github.io/portfolio-jules-landauer/`

## Pour modifier

- Les textes se modifient dans `index.html`.
- Les couleurs se modifient au début du fichier `assets/css/style.css`.
- Les fichiers PDF/ZIP peuvent être remplacés dans `assets/docs/`.
